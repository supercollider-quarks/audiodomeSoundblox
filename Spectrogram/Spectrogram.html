Spectrogram a spectral view

SpectrogramWindow a window like FreqScope



OSX only at the moment.



SuperCollider allows for various modes of graphical representations of sound. The frequency and wave scopes are useful and the Spectrogram adds yet another mode, the spectral view of sound. 


See also: FreqScope and Stethoscope



The Spectrogram comes in two modes, as a view that can be added to any SCWindow, and a special window that functions like the FreqScope window.



Creation / Class Methods


*new (parent, bounds, bufSize, color, background)

Short prose description of method.

parent - a Window (optional).

bounds - a Rect where the spectrogram appears in the parent (optional).

bufSize - frames in FFT window. default: 1024. Use power of 2 sizes, such as 512 or 2048.

color - The color drawn.

w = SCWindow.new("testing spectrogram", Rect(10,10, 600, 300)).front;

a = Spectrogram.new(w, Rect(101, 10, 400, 220));

a.start;

{Saw.ar(LFNoise1.kr(1).range(20,2250))}.play



Accessing Instance and Class Variables

rate_(arg1)

rate

Refresh rate. Default is 25 frames per second 

bufSize

frames in FFT window. see above.

intensity_(arg1)

intensity

Intensity is how strong the drawn color is.

Default value is 5.

Doing Some Task (optional)


A short bit of prose explaining something about the task.


start

starts the analysis and drawing.

// inline example

g = Spectrogram.start;

g.doSomething; // explanation

g.cleanUpMethod;



stop

stops analysis and drawing



inbus_ (inbus)

The audio bus of analysis.


color_ (argcolor)

The color that is drawn.

a = Spectrogram.new( color: Color.red);

{LPF.ar(WhiteNoise.ar(1), LFNoise1.kr(1).range(20,12250))}.play;

a.start;

a.start;

{AudioIn.ar(1)}.play // feedback warning!


a.intensity = 5;

a.rate = 25

a.color_(Color.red)
