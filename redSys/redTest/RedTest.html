RedTest generate test signals



*ar(amp, pan)

returns a SinOsc - stereo pseudo ugen

*sf(bus, group)

start a mono soundfile on bus+group

*sf2(bus, group)

start a stereo soundfile on bus+group

sfBus_(out)

set output bus for soundfile

<sfGrp

group for soundfile

sfFree

free buffer and synth for soundfile

*speaker(channels)

pink noise speaker test.  channels should be an array of out indices.

*speaker2(channels)

ping speaker test.  channels should be an array of out indices.

a= {RedTest.ar(MouseY.kr(1, 0), MouseX.kr(-1, 1))}.play //x= amp, y= pan

a.free;

a= {RedTest.ar(SinOsc.kr(1), SinOsc.ar(0.75))}.play

a.free;


a= RedTest.sf2(0, s.defaultGroup); //out bus and group

a.sfFree


a= RedTest.speaker([0, 1])

a.stop
