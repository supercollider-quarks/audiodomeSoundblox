RedHuffman huffman coding


"to use fewer bits for more frequently occurring character"

see http://www.cs.duke.edu/csed/poop/huff/info/


see also: RedBase64 RedLZ77 RedLZW RedRLE


*encode(string)

encodes a string with ascii characters into a string with binary numbers.

creates a tree and a dict as well.

*decode(string)

decodes a binary string into a string with ascii characters.

make sure you use the tree and dict generated by the encode method.

*binaryStringToBytes(string)

convert a binary string into 8-bit bytes.

up to 7 zeros might be padded automatically at the end.Â  the number of zeros is stored in pad.

pad is semi-private and used when converting back to bytes.

*bytesToBinaryString(array)

convert an array of 8-bit bytes into a binary string.

<>tree

multi-level array with characters as leaves.

<>dict

look-up dictionary with characters and binary strings.

<>pad

semi-private counter for number of padded zeros.

set by binaryStringToBytes and read by bytesToBinaryString.
