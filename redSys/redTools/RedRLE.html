RedRLE run-length encoding/decoding


see http://en.wikipedia.org/wiki/Run-length_encoding


see also: RedBase64 RedHuffman RedLZ77 RedLZW


*encode(array)

returns an array with length, value pairs.

*decode(array)

array must be length, value pairs.



//--

a= "WWWWWWWWWWWWBWWWWWWWWWWWWBBBWWWWWWWWWWWWWWWWWWWWWWWWBWWWWWWWWWWWWWW".ascii;

a.size;

b= RedRLE.encode(a);

b.size/a.size; //21% compressed

c= RedRLE.decode(b);

c.size;

a==c;

c.collect{|x| x.asAscii}.join;
