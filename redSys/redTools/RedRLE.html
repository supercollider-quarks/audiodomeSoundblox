RedRLE run-length encoding/decoding


see http://en.wikipedia.org/wiki/Run-length_encoding


see also: RedBase64 RedHuffman RedLZW


*encode(array)

returns an array with length, value pairs integers

*decode(array)

array must be length, value pairs integers

returns a string



//--

a= "WWWWWWWWWWWWBWWWWWWWWWWWWBBBWWWWWWWWWWWWWWWWWWWWWWWWBWWWWWWWWWWWWWW";

a.size;

b= RedRLE.encode(a);

b.size;

c= RedRLE.decode(b);

c.size;

a==c;


a= "JOEYNJOEYNJOEYJOEYNJOEYNJOEYJOEYNJOEYNJOEYJOEYNJOEYNJOEY";

a.size;

b= RedRLE.encode(a);

b.size; //performs really bad with non repeating data

c= RedRLE.decode(b);

c.size;

a==c;
