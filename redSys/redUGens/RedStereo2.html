RedStereo2 panning effect


with different LFOs


see also: RedStereo


*ar(in, rate, depth)

returns four panned bandpass filters - pseudo ugen

rate - default 0.2

depth - default 0.1



//--

s.boot

a= {RedStereo2.ar(In.ar(10))}.play

b= RedTest.sf2(10);

b.sfBus= 0; //plain

b.sfBus= 10; //redstereo - subtle difference


b.free;

a.free;


//--

a= {RedStereo2.ar(In.ar(10), MouseX.kr(0, 4), MouseY.kr(0, 1))}.play

b= RedTest.sf2(10);

b.sfBus= 0; //plain

b.sfBus= 10; //redstereo


b.free;

a.free;
